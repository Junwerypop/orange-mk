# Orange Market Platform

## Overview

Orange Market Platform is a web-based platform designed to connect orange farmers with buyers in a more transparent and efficient way.

The platform allows farmers to update the cultivation status of their orchards, while buyers can monitor orange availability, quality grades, and market information before making purchasing decisions.

This project is currently developed as an interactive prototype for demonstration and validation purposes.

---

## Problem Statement

Many orange farmers face difficulties in communicating production status and finding buyers at the right time.

Buyers often lack visibility into:

* Orchard locations
* Production readiness
* Orange varieties
* Quality grades
* Expected harvest periods

This platform aims to bridge the information gap between farmers and buyers.

---

## Features

### Farmer Features

* Register and manage orchard information
* Update cultivation status
* Display orchard location
* Manage orange varieties
* Indicate harvest readiness
* Upload orchard images

### Buyer Features

* Browse available orchards
* View orange varieties and grades
* Monitor market information
* Compare purchasing options
* Access orchard details and status updates

### Market Dashboard

* Market overview
* Orange classification
* Active buyers
* Pricing information
* Orchard map visualization

---

## Orange Growth Stages

The platform tracks orchard progress through several stages:

1. Land Preparation
2. Planting
3. Growth Stage
4. Flowering Stage
5. Fruit Development
6. Harvest Ready
